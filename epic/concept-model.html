Concept Model

Entities and Attributes

User

  • userId (primary key)
  • userActivationToken (for account verification)
  • userAgent
  • userAvatarUrl
  • userBlocked
  • userEmail
  • userHandle
  • userHash (for account password)
  • userIpAddress

User Detail

  • userDetailId (primary key)
  • userDetailUserId (foreign key)
  • userDetailAboutMe
  • userDetailAge
  • userDetailCareer
  • userDetailDisplayEmail
  • userDetailEducation
  • userDetailGender
  • userDetailInterests
  • userDetailRace
  • userDetailReligion

Question (Dan Questions)

  • questionId (primary key)
  • questionUserId (foreign key)
  • questionContent
  • questionValue

Answer

  • answerUserId (foreign key)
  • answerQuestionId (foreign key)
  • answerResult
  • answerScore

Match

  • matchUserId (foreign key)
  • matchToUserId (foreign key)
  • matchApproved

Report

  • reportUserId (foreign key)
  • reportAbuserID (foreign key)
  • reportAgent
  • reportContent
  • reportDate
  • reportIp

Relationships

  • One User has one User Detail
  • Many Users answer many Questions
  • Many Users can match with many Users
  • Many Users can report many Users
