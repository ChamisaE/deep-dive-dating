Concept Model

Entities and Attributes

User

  • userId (primary key)
  • userActivationToken (for account verification)
  • userAboutMe
  • userAge
  • userAvatarUrl
  • userEmail
  • userGender
  • userHandle
  • userHash (for account password)
  • userRace

User Detail

  • userDetailId (primary key)
  • userDetailUserId (foreign key)
  • userDetailIpAddress
  • userDetailUserAgent
  • userDetailDisplayEmail

Questionnaire (Dan Questions)

  • questionnaireId (primary key)
  • questionnaireUserId (foreign key)
  • questionnaireDrinker
  • questionnaireDrugs
  • questionnaireEducation
  • questionnaireKidsFuture
  • questionnaireKidsNow
  • questionnaireReligious
  • questionnaireSmoker

Question

  • questionUserId (foreign key)
  • questionQuestionnaireId (foreign key)
  • questionResult

Match

  • matchUserId (foreign key)
  • matchToUserId (foreign key)
  • matchApproved

Report

  • reportUserId (foreign key)
  • reportAbuserID (foreign key)
  • reportContent
  • reportDate

Relationships

  • One User has one User Detail
  • One User has one Questionnaire
  • One User has one Question
  • Many Users can match with many Users
  • Many Users can report many Users
