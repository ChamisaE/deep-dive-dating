Concept Model

Entities and Attributes

User

  • userId (primary key)
  • userActivationToken (for account verification)
  • userAboutMe
  • userAge
  • userAvatarUrl
  • userEmail
  • userGender
  • userHandle
  • userHash (for account password)
  • userRace

Questionnaire (Dan Questions)

  • questionnaireId (primary key)
  • questionnaireUserId (foreign key)
  • questionnaireDrinker
  • questionnaireDrugs
  • questionnaireEducation
  • questionnaireKidsFuture
  • questionnaireKidsNow
  • questionnaireReligious
  • questionnaireSmoker

Solo Match

  • soloMatchUserId (foreign key)
  • soloMatchToUserId (foreign key)

Pair Match

  • pairMatchId (primary key)
  • pairMatchFirstUserId (foreign key)
  • pairMatchSecondUserId (foreign key)
  • pairMatchResult

Report

  • reportId (primary key)
  • reportUserId (foreign key)
  • reportAbuserID (foreign key)
  • reportContent

Relationships

  • One User has one Questionnaire
  • One User can have many Solo Matches
  • One User can have many Pair Matches
  • One User can have many reports
