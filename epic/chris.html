Use Case

Title: Receives an email notification

Description: Chris receives an email notification from dating site URL, logs into site to check status updates

Name of the User and Their Role:Chris Smith--potential love match

Usage Preconditions: Must have internet capable device, must be logged in

Usage Post-conditions: Server sends a message to Chris' most recent match and saves his most recent Personality updates

Interaction Flow:

  • Chris receives an email notification from blamedandatedan.online, his new favorite dating website.
  • After work, at home, Chris opens his email account in Chrome to look at email and sees that it is a message from a new match!
  • After Chris reads the email he decides to respond so he enters blamedandatedan.online in Chrome.
  • Server receives the request and, since Chris is still logged in, returns Chris' profile page.
  • Chris looks to the menu on the left side of the page and finds the link for his Match List and clicks it.
  • Server receives the request and returns the page containing an array of the profiles Chris has liked.
  • Chris looks for the woman who just messaged him and notices that the heart next to her name is now illuminated; he clicks the link to her profile.
  • Server receives request and returns the page for Chris' match.
  • Chris looks through her page noticing that there is now new content to view including a message box. Chris enters a message and his information into the form and clicks submit.
  • Server receives request and sends an email notification to his match's email account.
  • Chris now decides to update his Personality section, so he looks to the menu again and clicks the link to his Personality editor page.
  • Server receives the request and returns the editor page.
  • Chris updates his Personality section with his likes and dislikes and clicks finish.
  • Server receives request and and saves the new information.
  • Chris uses the profile link in the menu to return to his page
  • Server receives request and returns Chris' profile page.
  • Chris can now see that his information has updated. He is happy with this so he closes the window but doesn't log out.

Frequency of Use: Daily
